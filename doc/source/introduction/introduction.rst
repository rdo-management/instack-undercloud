RDO-Manager Introduction
========================

RDO-Manager is an OpenStack Deployment & Management tool for RDO. It is based on
`OpenStack TripleO <https://wiki.openstack.org/wiki/TripleO>`_ and
`SpinalStack <http://spinal-stack.readthedocs.org/en/latest/>`_ projects.
